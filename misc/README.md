## `misc/` — non-core utilities

This folder contains **one-off scripts, experiments, and prototypes** that are useful
for development but are **not part of the core ingestion pipeline**.

### Contents

- `db_connect_test.py`: quick connectivity check to Supabase Postgres.
- `testapi.py`: simple API-Football request test.
- `tools/streamlit_redis_viewer/`: Streamlit prototype UI for viewing fixture data from Redis.
- `obsolete/`: deprecated code kept for reference (not used by the current pipeline).
